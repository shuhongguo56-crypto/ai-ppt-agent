# AI PPT Agent visual asset library

This directory is the project-owned presentation primitive library. It stores
composition, shape, chart, icon and image-placement metadata rather than
copying finished third-party templates.

Design-system research references:

- PPT Master (`hugohe3/ppt-master`, MIT): layout-first validation, reusable
  layout/chart/icon primitives and overlap checks.
- PPTAgent (`icip-cas/PPTAgent`, MIT): structured presentation planning and
  iterative quality review.
- Presenton (`presenton/presenton`, Apache-2.0): canonical deck data and
  editable PPTX export.
- PptxGenJS (`gitbrent/PptxGenJS`, MIT): native Open XML presentation output.
- Real-ESRGAN (`xinntao/Real-ESRGAN`, BSD-3-Clause): local raster upscaling.

All in-product layouts and shape recipes are original project assets. External
photographs and generated images retain per-slide provenance in asset sidecars
and speaker notes.
